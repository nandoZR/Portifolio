AI Generator Image

Projeto Individual - HTML,CSS,JS
Uma aplicação web que utiliza IA para gerar imagens com base em prompts de texto inseridos pelos usuários. O projeto inclui uma interface atraente com campos para inserir prompts, botões de geração e exibição de imagens geradas. Os usuários podem fazer o download das imagens criadas.A API utilizada foi a do OpenJourney.
